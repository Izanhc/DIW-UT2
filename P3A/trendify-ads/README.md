# TrendifyAds Project

Este proyecto ha sido generado con Vite y SASS siguiendo las especificaciones de la Práctica 3.1.

## Instrucciones de Instalación

Debido a que no se detectó `npm` en el entorno actual, no se pudieron instalar las dependencias automáticamente. Por favor, sigue estos pasos:

1.  Abre una terminal en la carpeta del proyecto:
    ```bash
    cd trendify-ads
    ```
2.  Instala las dependencias:
    ```bash
    npm install
    ```
3.  Inicia el servidor de desarrollo:
    ```bash
    npm run dev
    ```
4.  Para construir el proyecto para producción (GitHub Pages):
    ```bash
    npm run build
    ```
    Los archivos generados estarán en la carpeta `docs`.

## Estructura del Proyecto

- `src/`: Código fuente.
  - `styles/`: Archivos SASS organizados en parciales.
  - `assets/`: Imágenes y recursos.
  - `index.html`: Página principal (Landing Page).
  - `login.html`: Página de inicio de sesión.
- `vite.config.js`: Configuración de Vite para múltiples páginas y salida en `docs`.
